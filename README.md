# Fragrance Explorer

Una aplicación web interactiva para explorar y analizar una base de datos de fragancias.

## Características

- 🔍 Búsqueda y filtrado avanzado de fragancias
- 📊 Dashboard con análisis y estadísticas
- 🎨 Interfaz moderna y responsive
- 📱 Visualización detallada de cada fragancia

## Tecnologías

- HTML5, CSS3, JavaScript
- Chart.js para visualizaciones
- SheetJS para manejo de datos

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/TU_USUARIO/fragrance-explorer.git

# Navegar al directorio
cd fragrance-explorer

# Instalar dependencias
npm install

# Abrir index.html en tu navegador
```

## Uso

Simplemente abre el archivo `index.html` en tu navegador web favorito. La aplicación funciona completamente del lado del cliente sin necesidad de servidor.
